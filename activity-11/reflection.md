**Key Learnings** 
- I learned how breaking code into smaller reusable components makes applications easier to organize and maintain. 
- React components, props, state management using useState.

**Comparison** 
The React version was much cleaner and easier to manage compared to vanilla Javascript. In vanilla JavaScript, I had to manually update the DOM, track variables, and handle UI change. In React, the UI automatically updates whenever the state changes, which reduces repetitive code, and the React simply looks much more organize compare to Javascript.

**Challenges**
One of the most difficult concepts to understand was how state works in React and how components re-render when the state changes. Understanding props and passing data between components also took some practice.
